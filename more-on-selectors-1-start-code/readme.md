# Class 선택자와 ID 선택자

## Class 선택자
- 재사용이 가능하다.
- 스타일링 목적으로 사용된다.
- 가장 많이 사용되는 선택자이다.
```
.some-class { ... } 

<div class="some-class"></div>
<p class="some-class"></p>
```



## ID 선택자
- 페이지당 한번만 사용된다.
- 한 요소에만 영향을 줘야하는 경우 사용된다.
- 스타일 적용만을 위한 사용을 권장하지 않는다.
```
#some-id { ... }

<div id="some-id"></div>
<p><p>
```
